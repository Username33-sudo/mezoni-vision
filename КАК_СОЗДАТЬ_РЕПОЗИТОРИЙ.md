# Как создать репозиторий на GitHub

## Шаг 1: Создайте репозиторий

1. Откройте https://github.com/new
2. Введите имя: `mezoni-vision` или `mezoni-vision-standalone`
3. Описание: "Система анализа заполненности стеллажей по скриншотам из Twin"
4. Выберите **Public** или **Private**
5. **НЕ нажимайте** "Initialize this repository with README" (у нас уже есть свой)
6. Нажмите **Create repository**

## Шаг 2: Инициализируйте Git локально

Откройте терминал в папке проекта:

```bash
cd "C:\Users\кеша\Desktop\Новая папка\mezoni-vision-standalone"

# Инициализация Git
git init

# Добавляем все файлы
git add .

# Первый коммит
git commit -m "Initial commit: Mezoni Vision v1.0

- FastAPI сервер для анализа скриншотов
- OpenCV анализ заполненности рядов
- Веб-интерфейс для загрузки файлов
- Скрипт тестирования с логированием
- Standalone версия (exe)"
```

## Шаг 3: Привяжите репозиторий

```bash
# Замените YOUR_USERNAME на ваш логин GitHub
git remote add origin https://github.com/YOUR_USERNAME/mezoni-vision.git

# Переименуйте ветку в main
git branch -M main

# Отправьте на GitHub
git push -u origin main
```

## Шаг 4: Проверьте

Откройте ваш репозиторий на GitHub — должны появиться все файлы.

## Дополнительная настройка

### Добавьте темы (topics)

На странице репозитория нажмите ⚙️ рядом с "About" и добавьте:
- `computer-vision`
- `warehouse`
- `fastapi`
- `opencv`
- `twin`
- `analytics`

### Включите GitHub Pages (опционально)

Если хотите показать демо:
1. Settings → Pages
2. Source: GitHub Actions
3. Добавьте workflow для деплоя

### Добавьте Issues шаблоны

Создайте `.github/ISSUE_TEMPLATE/bug_report.md` для баг-репортов.

## Если что-то пошло не так

### "fatal: remote origin already exists"
```bash
git remote remove origin
git remote add origin https://github.com/YOUR_USERNAME/mezoni-vision.git
```

### "failed to push some refs"
```bash
git pull origin main --rebase
git push origin main
```

### Случайно запушили .env с ключами?
1. Срочно смените API ключ в Gemini
2. Удалите файл из истории:
```bash
git filter-branch --force --index-filter \
  "git rm --cached --ignore-unmatch .env" \
  --prune-empty --tag-name-filter cat -- --all
git push origin --force --all
```
