# Mezoni Vision

Система анализа заполненности стеллажей склада по скриншотам из BIM-модели **Twin**.

## Возможности

- Анализ скриншотов стеллажей (до 4 секций одновременно)
- Определение заполненности рядов по цветам:
  - 🟢 **Свободно** — зелёный цвет
  - 🟡 **Средне** — жёлтый цвет
  - 🔴 **Полно** — красный цвет
- Расчёт средней заполненности склада
- Рекомендации по размещению товаров
- Веб-интерфейс для загрузки скриншотов
- API для интеграции с другими системами
- Логирование тестов для валидации точности

## Быстрый старт

### Запуск сервера

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск сервера
python src/main.py
```

Сервер запустится на `http://localhost:8000`

### Использование

1. Откройте `http://localhost:8000` в браузере
2. Перетащите скриншоты из Twin (1-4 файла)
3. Нажмите "Анализировать"
4. Получите результат с рекомендациями

## Режимы анализа

### OpenCV (по умолчанию)

Анализ на основе цветовых порогов в HSV-пространстве. Быстро, работает офлайн.

```bash
# В .env файле:
ANALYSIS_MODE=opencv
```

### Google Gemini AI (опционально)

Более точный анализ с помощью ИИ. Требует API ключ.

```bash
# В .env файле:
ANALYSIS_MODE=gemini
GEMINI_API_KEY=your_api_key_here
```

## Тестирование

Для валидации точности анализа используйте скрипт `log_test.py`:

```bash
python log_test.py
```

Скрипт:
1. Проверяет доступность сервера
2. Принимает пути к скриншотам
3. Отправляет на анализ
4. Сравнивает с ручным подсчётом
5. Сохраняет результаты в `test_log.json` и `test_results.txt`

### Критерии успеха

| Метрика | Значение |
|---------|----------|
| Точность по свободным рядам | ≥ 90% |
| Время анализа | < 1 минуты |
| Понятный интерфейс | Да |

## API Endpoints

### `GET /`
Веб-интерфейс для загрузки скриншотов

### `POST /analyze`
Анализ загруженных изображений

**Request:**
```
photo1: File (image)
photo2: File (image, optional)
photo3: File (image, optional)
photo4: File (image, optional)
```

**Response:**
```json
{
  "success": true,
  "fill_stats": {
    "total_rows": 15,
    "green": 5,
    "yellow": 3,
    "red": 7,
    "avg_fill_pct": 62.0
  },
  "recommend": "Есть свободные ряды (5). Направлять на свободные места."
}
```

### `GET /health`
Проверка статуса сервера
